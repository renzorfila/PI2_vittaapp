# PI2 VittaApp

Sistema web desenvolvido para gerenciamento de agendamentos profissionais.

## Tecnologias utilizadas

### Backend
- Java 21
- Spring Boot
- MySQL
- Maven

### Frontend
- React
- Vite
- JavaScript

## Estrutura do projeto

```txt
PI2_Vittaapp/
 ├── vittaapp_backend/
 └── vittaapp_frontend/
```

## Como executar

### Backend

```bash
cd vittaapp_backend
mvn spring-boot:run
```

### Frontend

```bash
cd vittaapp_frontend
npm install
npm run dev
```

## Funcionalidades

- Cadastro de usuários
- Perfis profissionais
- Agendamentos
- Disponibilidade de horários
